## 天气预报cli 工具 

- cli开发的数据来源[中国天气网](http://www.weather.com.cn/)

## 使用方法 

```
npm i wci-cli-tool
```

```
weather -h 
weather -c 成都 -t -d 1 
weather -c 成都 -j -d 1 --hourly
weather -c 成都 -t -d 7 --hourly --date 11-11
weather -c 成都 -t -d 15
weather -c 成都 -t -d 40 
```
